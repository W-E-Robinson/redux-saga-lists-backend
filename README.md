# redux-lists-backend

Backend simple Node.js for redux-lists, see [redux-lists-docker](https://github.com/W-E-Robinson/redux-lists-docker) for more details on running.

## API Reference

#### Get list

```http
GET /list
```
#### Patch item

```http
PATCH /list/:id
```
#### Post new item

```http
POST /list
```

#### Delete item

```http
GET /list/:id
```
